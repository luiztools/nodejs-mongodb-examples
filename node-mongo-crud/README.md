# node-mongo-crud

A simple MongoDB CRUD example with Node.js.

## How to Run

1. Install Node.js and download MongoDB Community Server
2. run a mongod server at port 27017
3. cd node-mongo-crud
4. copy .env.example as .env
5. fill the .env
6. npm install
7. npm start
